# RabbitMQ_php
 PHP + RabbitMQ with adv feture

## install libamqp

```bash
brew install rabbitmq-c
brew install rabbitmq
brew install amqp-cpp
```

> ubuntu

```bash
apt install libssl-dev librabbitmq-dev
```



## install ext

```bash
pecl install amqp
```

## Install php-amqplib

```bash
composer require php-amqplib/php-amqplib
```
